azimuthdp back to ToC or Data Property ToC

IRI: http://cdm.ovh/rsm/topology/topology#azimuth

outward azimuth at a port (of a linear element). "outward" means, seen in the direction of leaving the linear element. azimuth value is given with respect to North in a WGS84-based reference system. Value is in degrees, in the range [-180, +180] as per rdflib convention, with 0 pointing to North.
has domain
Linear Element c
has range
decimal
